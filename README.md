# 🍕 La Piazza – Authentic Italian Pizza Restaurant

A professional Pizza Restaurant website built with HTML, CSS, and JavaScript.  
Recipes are fetched dynamically from the **DummyJSON Recipes API**.

## 🌐 Live Demo
> Add your GitHub Pages link here after deployment

## 🛠️ Tech Stack
- **HTML5** – Semantic structure
- **CSS3** – Custom properties, Grid, Flexbox, Animations
- **JavaScript (ES6+)** – Fetch API, DOM manipulation
- **DummyJSON API** – `https://dummyjson.com/recipes`

## 📁 File Structure
```
pizza-website/
├── index.html              # Main page
├── style.css               # All styles
├── app.js                  # API calls & interactivity
├── README.md
└── .github/
    └── workflows/
        └── deploy.yml      # GitHub Actions CI/CD
```

## ✨ Features
- 🔥 Hero section with animated intro
- 📖 About section with restaurant story
- 🍕 Recipe cards fetched live from DummyJSON API
- 🔍 Search recipes by name
- 📄 Modal popup with full recipe details (ingredients, instructions, nutrition)
- 📱 Fully responsive design
- ⚡ Load More pagination
- 🚀 Auto-deployed via GitHub Actions → GitHub Pages

## 🚀 Deployment (CI/CD)
Every push to `main` triggers GitHub Actions to deploy to GitHub Pages automatically.
